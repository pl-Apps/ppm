# ppm
pl Package Manager

## Install packages
```sh
ppm install [package name] 
```

## Remove packages 
```sh
ppm remove [package name]
ppm uninstall [package name]
```

## Update packages
```sh
ppm update [package name]
```

## Search packages
```sh
ppm search [package name]
```

## Get the all installed packages
```sh
ppm list
```
